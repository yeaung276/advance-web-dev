Installation & Setup

The platform uses uv for dependency management and execution. Follow the steps below to run the system locally.

  1. Install all project dependencies: 
    uv install
  2. Create and migrate the database schema: 
    uv run manage.py migrate
  3. Insert all required initial data: 
    uv run scripts/bulk_insert.py
  4. Create super user if you create your own fresh sqlite database: 
    uv run manage.py createsuperuser
  5. Login to admin panel through admin and create normal user for API mutation access. this step is optional. You can use admin credential you created in the earlier step.
  6. Start the development server: 
    uv run manage.py runserver

Original Data Source

The data used in this platform is derived from the Open Supernova Catalog (OSC), a community-driven compilation of supernova observations from multiple surveys and publications.

Open Supernova Catalog (OSC)

OSC aggregates heterogeneous supernova datasets into a unified, machine-readable format, enabling large-scale analysis and cross-survey comparison.
